🏆 Egyptian League Management System
A robust, Desktop-based application built with Java and Object-Oriented Programming (OOP) principles to streamline the management of football leagues. This project demonstrates the practical application of software engineering concepts in a real-world sports management scenario.
🚀 Key Features
• Team & Player Management: Efficiently add, update, and manage football teams and player rosters.
• Match Scheduling: Automated tools for scheduling league matches and tracking results.
• Point Calculations: Real-time, automated league table updates based on match outcomes.
• Live Statistics: Detailed dashboard providing insights into top scorers, team performance, and standings.
• Persistence: Integrated Local CSV storage to ensure data is saved and retrieved seamlessly between sessions.
🛠️ Tech Stack
• Language: Java.
• GUI Framework: Java Swing.
• Data Storage: Flat-file CSV for lightweight data persistence.
• Architecture: Strictly follows OOP principles (Inheritance, Encapsulation, Polymorphism).
💻 How to Run
1.	Clone the repository to your local machine.
2.	Open the project in your preferred IDE (NetBeans, IntelliJ, or Eclipse).
3.	Run the Main.java file to launch the GUI.
